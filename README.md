# 🍔 Food Ordering System

A full-stack web application that allows users to browse restaurants, add food items to cart, and place orders online.  
This system includes both user and admin functionalities with a clean and responsive interface.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Browse Restaurants & Food Items
- Add to Cart
- Place Orders
- View Order History

### 🛠 Admin Module
- Add / Update / Delete Restaurants
- Manage Food Items
- View All Orders
- Manage Users
